## 웹사이트 개요
**오픈소스 소프트웨어 수업 기말고사 대비 정리** 학습 웹사이트입니다.

## 주요 내용
- **PART 1**: Git 기초 설정
- **PART 2**: 3대 영역 (Working Directory, Staging, Repository)
- **PART 3**: 브랜치와 원격 저장소
- **PART 4**: 고급 기능 (Reset vs Revert, 라이선스 등)

## 기술 구성
- HTML/CSS/JavaScript
- 다크 테마 + 반응형 디자인
- 사이드바 네비게이션
- 코드 복사 기능

# 웹사이트 기능 소개

## 1. 사이드바 네비게이션
- 웹사이트 왼쪽에 위치한 사이드바를 통해 각 섹션으로 부드럽게 이동할 수 있습니다.
- 현재 보고 있는 섹션에 따라 네비게이션 링크가 활성화되어 표시됩니다.
